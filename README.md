# Systems Specialist Developer Portfolio

A high-performance, responsive, and animated portfolio specifically designed for software developers focusing on complex enterprise systems like MLM (Multi-Level Marketing), POS (Point Of Sale), and School Management Systems (LMS).

## Features

- **Systems-First Design**: Industrial aesthetic focusing on reliability and scalability.
- **Dynamic Projects**: Specialized modules for MLM compensation logic, POS infrastructure, and Educational platforms.
- **Full Interactivity**: 
    - Custom animated "Glass" cursor.
    - Scroll-reveal animations using Framer Motion.
    - Typewriter effects for system roles.
    - Smooth interactive navigation.
- **Glassmorphism**: Modern tech-heavy UI style using Tailwind CSS and backdrop filters.
- **Data-Driven**: All content is managed in a single `data.js` file for easy updates.

## Tech Stack

- **React 18**: Frontend framework.
- **Vite**: Ultra-fast build tool.
- **Tailwind CSS**: Utility-first styling.
- **Framer Motion**: Production-ready animation library.
- **Lucide React**: Clean and consistent iconography.
- **React Simple Typewriter**: Header animations.

## Prerequisites

- Node.js (v16.0 or higher)
- npm or yarn
